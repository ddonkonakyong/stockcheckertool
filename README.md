# Stock Analysis Tool

A comprehensive stock analysis dashboard built with Python and Streamlit.

## Features
- **Interactive Charts:** Visualize historical price data with SMA overlays.
- **Technical Analysis:** RSI and MACD indicators.
- **Valuation (DCF):** Automated Discounted Cash Flow analysis with WACC estimation.
- **Multi-Market Support:** Analyze stocks from US and Korean markets (KOSPI/KOSDAQ).
- **Latest News:** Live news feed for selected tickers.

## Setup
1. Install dependencies:
   ```bash
   pip install -r requirements.txt
   ```
2. Run the application:
   ```bash
   python -m streamlit run src/main.py
   ```
